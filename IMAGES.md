# Images à fournir — Cercle Partenaire Urbafence

La page attend **12 images** dans le dossier `images/`. Tant qu'un fichier n'est pas présent, un **placeholder gris/vert s'affiche automatiquement** à la place (repli `onerror`) : la page n'est donc jamais « cassée ».

## Comment ajouter une image

1. Enregistrez l'image choisie dans le dossier `images/` **avec exactement le nom indiqué** dans le tableau ci-dessous (ex. `hero-principal.jpg`).
2. C'est tout : rechargez la page, l'image remplace le placeholder.
3. (Optionnel) Si vous changez un nom de fichier, pensez à le reporter dans `index.html` (cherchez l'identifiant `IMG-...` écrit en commentaire au-dessus de chaque image).

> Conseil : respectez les ratios indiqués pour éviter que l'image soit déformée ou rognée. Format `.jpg` recommandé (ou `.webp` pour des fichiers plus légers — pensez alors à adapter l'extension dans `index.html`).

## Note sur les motifs du carrousel

Les motifs (Tocima, Octopus, Agostine Magma, Artefact, Opale Grio) ne sont **pas des photos** : ce sont des dessins vectoriels (SVG) intégrés directement dans le code. Vous n'avez donc pas d'image à fournir pour eux. Leurs liens pointent provisoirement vers `https://urbafence.com/`.
